Create a function with namend parameters With this function it is easy to create Lua functions with default values for their arguments and defining names for them to circumvent the position dependentness of the arguments. This is especially useful for functions with a lot of arguments.

Attention
When calling with named parameters, you have to give a table with the named arguments as the only parameter to the created functions.

Use as:

myfunction = util.CreateFancyFunction(
{{"a"},{"b",7},{"c",5}},
function(a, b, c)
print(a, b, c)
end
)
myfunction( 3 ) -->> prints: 3 7 5
myfunction( 4, 2 ) -->> prints: 4 2 5
myfunction{ a=3, c=1 } -->> prints: 3 7 1
myfunction( { a=3, c=1 } ) -->> prints: 3 7 1
