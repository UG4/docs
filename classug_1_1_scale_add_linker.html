Detailed Description

template<typename TData, int dim, typename TDataScale, typename TRet = TData>
class ug::ScaleAddLinker< TData, dim, TDataScale, TRet >

This linker recombines the data like

l = s_1 * i_1 + ... + s_k * i_k

where, l, i_1, ..., i_k are of the same data type and the s_i are some scaling factor of a (possibly) different type, that is applicable to the data type

Template Parameters
TDataexported and combined Data type
dimworld dimension
TDataScaletype of scaling data
