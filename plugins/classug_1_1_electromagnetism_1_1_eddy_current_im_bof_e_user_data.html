Detailed Description

template<typename TGridFunc>
class ug::Electromagnetism::EddyCurrentImBofEUserData< TGridFunc >

UserData based class for the computation of the imaginary part of the magnetic field \( \mathbf{B} = \frac{i}{\omega} \mathbf{rot} \, \mathbf{E} = - \frac{1}{\omega} \mathbf{rot} \, \mathrm{Re} \, \mathbf{E} + i \frac{1}{\omega} \mathbf{rot} \, \mathrm{Im} \, \mathbf{E} \), i.e. the vector field \( \mathrm{Im} \, \mathbf{B} = \frac{1}{\omega} \mathbf{rot} \, \mathrm{Re} \, \mathbf{E} \).
