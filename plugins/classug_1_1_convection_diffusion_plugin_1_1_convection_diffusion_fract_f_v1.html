Detailed Description

template<typename TDomain>
class ug::ConvectionDiffusionPlugin::ConvectionDiffusionFractFV1< TDomain >

Discretization for the Convection-Diffusion Equation in fractures.

This class implements the IElemDisc interface to provide element local assemblings for the convection diffusion equation in the low-dimensional fractures. using the degenerated elements. This module is complementary to ConvectionDiffusionFV1. Cf. ConvectionDiffusionBase base class for the problem setting.

See also
ConvectionDiffusionBase for the problem settings
ConvectionDiffusionFV1 for the FV discretization in the full-dim. domain
