inline

Create classic Richards equation.

Element discretization for: \frac{\partial S(-h)}{\partial t} + \nabla \cdot [- C \nabla (h+z)] = 0

Input parameters: saturation S, conductivity C = Cabs*Crel, gravity $\nabla z$

Create elem disc.
