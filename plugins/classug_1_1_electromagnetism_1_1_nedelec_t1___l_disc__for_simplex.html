Detailed Description

template<typename TDomain, typename TElem>
class ug::Electromagnetism::NedelecT1_LDisc_forSimplex< TDomain, TElem >

Helper class for the specialization of NedelecT1_LDisc for simplices (triangles and tetrahedrons)

This class template implements the Whitney-1 (Nedelec) elements for triangles and tetrahedrons (i.e. implements the classical variant of these elements),

See also
NedelecT1_LDisc.

Consider a grid element \(T\). Let \(w^{(0)}_i\) denote the linear (Lagrangian) shape functions equal to 1 at corner i and to 0 at the other corners of \(T\). These are the Whitney-0 shape functions. The Whitney-1-shape functions \(w^{(1)}_e\) (constituting the Nedelec element) are assigned to edges \(e\) of \(T\). For the edge \(e = (i, j)\), this function is given by formula

\[ w^{(1)}_e = w^{(0)}_i \mathbf{grad} w^{(0)}_j - w^{(0)}_j \mathbf{grad} w^{(0)}_i. \]

These are vector-valued shape functions. They are well-defined for simplices (i.e. for triangles in 2d and tetrahedra in 3d), whereas the corresponding physical problems are typically formulated only in 3d.

Functions \(w^{(1)}_e\) are used for discretization of the differential operators of type \(\mathbf{rot} \mu^{-1} \mathbf{rot}\) with the discretization error \( O(h) \). The physical parameters (like \( \mu \)) are then considered constant over \(T\), so that this tool kit sets the physical factors to 1. (The local matrices can be multiplied by these factors later.) The local stiffness matrix computed by this tool kit is

\[ (S_{h,T})_{e_1,e_2} = \int_T \mathbf{rot} w^{(1)}_{e_1} \cdot \mathbf{rot} w^{(1)}_{e_2} dx, \]

and the mass matrix is

\[ (M^{(1)}_{h,T})_{e_1,e_2} = \int \sigma w^{(1)}_{e_1} \cdot w^{(1)}_{e_2} dx. \]

The computation of the stiffness matrix is exact. (The integrand is a linear function.) But the computation of the mass matrix is implemented by an inexact quadrature formula.

Remarks
Note that the definition of the Whitney-1 function depends on the ordering of the nodes in the element. More precisely, for a different ordering, the shape function \(w^{(1)}_e\) may have a different sign. Thus, the ordering should be consistent in all the elements: If two elements \(T_1\) and \(T_2\) share the edge \(e\) with the ends at corners \(i\) and \(j\) (that are shared, too) then this edge should be \(e = (i, j)\) for both the elements (and not for example \(e = (i, j)\) for \(T_1\) but \(e = (j, i)\) for \(T_2\)). In all the other respects, the ordering does not play any essential role. In particular, the singes of the degrees of freedom in the solution depend on the global ordering of the corners (but the absolute values of these dofs are invariant). For this, the solution of the discretized system may be only considered in connection with the given ordering of the corners (in the edges).

References:

  • O. Sterz. Modellierung und Numerik zeitharmonischer Wirbelstromprobleme in 3D. PhD thesis, Heidelberg University, 2003.
  • A. Bossavit. Computational Electromagnetism. Academic Press (Boston), 1998

gets the correspondence between the edge dof indices and the corners of the element

gets the correspondence between the edge dof indices and the corners of the element

The edge dof indices correspond to the order of the edges in the reference element. To every edge, two vertices (the corners of the element) are assigned. We need them to be ordered independently of the element. This function assignes to every edge (i.e. every edge dof index) two indices of the corners of the element in order that corresponds to the global order of the vertices associated with these corners.

assembles the discrete weak div operator

This function assembles the local weak divergence operator \( \mathbf{B} \):

\[ \mathbf{B}_{v,e} = - \int_T w^{(1)}_e \cdot \mathbf{grad} w^{(0)}_v \, dx, \]

where \(T\) is the element, \(v\) its corner and \(e\) its edge. The integration is perfomed by the midpoint rule which is exact because the integrand is linear. Thus, for \(e = [v_1, v_2]\)

\[ \mathbf{B}_{v,e} = - \frac{|T|}{N} \mathbf{grad} w^{(0)}_v \cdot (\mathbf{grad} w^{(0)}_{v_2} - \mathbf{grad} w^{(0)}_{v_1}) \, dx. \]

(with \(N\) being the number of corners of \(T\)).
