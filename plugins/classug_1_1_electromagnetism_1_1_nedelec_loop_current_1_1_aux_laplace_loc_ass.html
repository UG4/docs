Detailed Description

template<typename TDomain, typename TAlgebra>
class ug::Electromagnetism::NedelecLoopCurrent< TDomain, TAlgebra >::AuxLaplaceLocAss

Class for local assembling of the auxiliary Laplace operator.

This class assembles the local discretization of the Laplace operator in the closure of the subdomain of the source. (Out of this closure, the constraint of class OutOfSource is used.)
