External Libraries

Overview

ug4 uses some external libraries. Some of them are inculded in the ug4 sources, some have to be downloaded and compiled separately.

Here is a short overview of the libraries included in the sources. All those libraries have a license, which allows to use and distribute them even for commercial purposes.

  • linenoise (BSD-like License)
  • Lua (BSD-like License)
  • Metis explicitly licensed for use with <strong>ug4</strong>
  • ParMetis explicitly licensed for non-commercial use with <strong>ug4</strong>
  • lineintersect_utils (custom license)
  • RapidXML (Boost Software License - Version 1.0)
  • Shiny Profiler (zlib/libpng License)
  • tribox (custom license)
  • tritri (CHECK LICENSE)

Libraries which can be used by ug4, but are not part of the source-distribution due to license restrictions or due to their size are the following:

  • boost (Boost Software License)
  • mpi (different licenses)
  • tetgen (free for non-commercial use)
