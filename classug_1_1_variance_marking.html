Detailed Description

template<typename TDomain>
class ug::VarianceMarking< TDomain >

Marks elements above \( \theta * (\mu + width * \sigma) \) for refinement where \( \mu = E[\eta^2], \sigma^2 = Var[\eta^2] \)
